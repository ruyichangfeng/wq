    3.1.1个人审核模板消息：{state}替换审核状态(成功;失败)、{staff_name}替换申请人姓名、{staff_mobile}替换申请人手机号码、{explain}替换备注说明、{stime}替换审核时间；
    3.1.2公司审核模板消息：{state}替换审核状态(成功;失败)、{company_name}替换申请公司名称、{staff_mobile}替换负责人手机号码、{explain}替换备注说明、{stime}替换审核时间；
    3.1.3商铺审核模板消息：{state}替换审核状态(成功;失败)、{merchant_name}替换申请商铺名称、{merchant_address}替换商铺地址、{explain}替换备注说明、{stime}替换审核时间；
    3.2订单推送模板消息：{ordernumber}替换订单编号、{order}替换服务类别、{ftime}替换服务时间、{fadr}替换服务地点、{username}替换预约所填联系人、{fmobile}替换预约所填电话、{money}替换预计价格、{nickname}替换用户昵称、{addtime}替换下单时间；
    3.3抢单模板消息：可用{staff_name}替换服务人员姓名，{staff_mobile}替换服务人员手机，{offer}替换服务人员报价；
    3.4服务完成模板消息：{suremoney}替换服务人员确认价格；

    4.3用户竞价模板消息：{ordernumber}替换订单编号、{realname}替换下单人姓名、{mobile}替换下单人手机号码、{address}替换送货地址、{goods}替换商品、{money}替换预期金额、{size}替换购买数量、{gettime}替换希望送达时间、{addtime}替换发布时间；
    4.3商铺抢单模板消息：同竞价模板消息相同，且{offer}替换报价，{merchant_name}替换商铺名称；
